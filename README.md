## 🚀 安装

### 下载安装包

在 [Release](https://github.com/codepoolmax/CodePool-main/releases) 中根据平台选择对应的安装包进行安装即可。

#### Windows
```bash
# 下载 .exe 安装包
CodePool-Setup-x.x.x.exe
```

#### macOS
```bash
# 下载 .dmg 安装包
CodePool-x.x.x.dmg
```

#### Linux
```bash
# 下载 .AppImage 或 .deb 安装包
CodePool-x.x.x.AppImage
CodePool-x.x.x.deb
```

## 💡 使用

<div align="center">
<img src="public/软件首页 (1).png" alt="软件首页" width="700"/>
</div>

### 1. 启动软件

安装完成后，启动 CodePool 软件。

### 2. 注册登录

<div align="center">
<img src="public/取号界面 (2).png" alt="取号界面" width="600"/>
</div>

在软件中完成注册和登录。

### 3. 选择 IDE

选择您需要使用的 IDE（Cursor、Windsurf、Augment 等）。

### 4. 一键续杯

点击"取号"按钮，即可获取新的账号，享受无限续杯服务。

### 5. 开始编程

将获取的账号配置到对应的 IDE 中，开始您的高效 AI 编程之旅！
